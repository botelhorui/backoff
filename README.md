Install
-------

`backoff` uses [SBT][1] for building and requires Java 8.

```
$ git clone https://github.com/ist-dsi/backoff.git

$ cd backoff

$ sbt update

$ sbt compile
```

Then you can run the tests simply by:
`sbt test
